## 👉 คุณคือใคร? เปิดคู่มือของคุณ

| บทบาท (Role) | คุณทำอะไร | คู่มือ |
|-------------|-----------|--------|
| **ผู้ขอเบิกวัสดุ** | ขอเบิกวัสดุไปใช้หน้างาน ติดตามสถานะ | [requester.md](requester.md) |
| **ผู้อนุมัติ** | อนุมัติ/ปฏิเสธคำขอเบิก | [approver.md](approver.md) |
| **เจ้าหน้าที่คลัง** | รับ/จ่าย/โอน/นับสต๊อก ดูแลข้อมูลวัสดุ | [warehouse.md](warehouse.md) |
| **ฝ่ายจัดซื้อ** | สร้าง/จัดการใบสั่งซื้อ (PO) | [purchasing.md](purchasing.md) |
| **ผู้บริหาร / ผู้จัดการ** | ดูรายงานการเงิน ต้นทุน จัดการโครงการ | [executive.md](executive.md) |

## พื้นฐานที่ทุกคนควรรู้ (สรุปสั้น)

**เข้าสู่ระบบ:** กรอกอีเมล + รหัสผ่านที่ได้รับ · ใส่รหัสผิดเกิน 10 ครั้ง/15 นาที จะถูกระงับชั่วคราว

**เปลี่ยนรหัสผ่านตัวเอง:** กดชื่อตัวเองมุมล่างซ้าย → **บัญชีของฉัน** (ทุกบทบาททำได้) · ลืมรหัส → ให้ Admin ตั้งใหม่ให้

**สิทธิ์แบ่ง 2 มิติ:**
- **Role** = ทำอะไรได้ (ขอเบิก/อนุมัติ/คลัง/จัดซื้อ/บริหาร)
- **Access Level** = เห็นข้อมูลแค่ไหน (STAFF ไม่เห็นราคา · MANAGER เห็นต้นทุน+โครงการ · ADMIN เห็นทุกอย่าง)

**กฎที่ใช้ทั้งระบบ:**
- จำนวนวัสดุต้องเป็น**จำนวนเต็ม**เสมอ (จำนวนเงินใส่ทศนิยมได้)
- **สต๊อกติดลบไม่ได้**
- ทุกการเคลื่อนไหวถูกบันทึกย้อนแก้ไม่ได้ ตรวจสอบย้อนกลับได้
- เมนูแสดงตามสิทธิ์ — เห็นไม่ครบเหมือนเพื่อนเป็นเรื่องปกติ
