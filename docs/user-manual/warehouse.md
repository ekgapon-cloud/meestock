# คู่มือสำหรับ เจ้าหน้าที่คลัง (Warehouse)

> บทบาทของคุณ: **ดูแลของเข้า-ออกคลัง** — รับวัสดุ จ่ายวัสดุ โอนย้าย นับสต๊อก และดูแลข้อมูลวัสดุ/หมวดหมู่
> เมนูที่คุณใช้: Dashboard · Materials · หมวดหมู่วัสดุ · คำขอเบิกวัสดุ · รับวัสดุ · โอนย้ายระหว่างคลัง · นับสต๊อก · รายงาน · บัญชีของฉัน

---

## เริ่มต้นใช้งาน
1. เปิดเว็บระบบ → กรอก **อีเมล** และ **รหัสผ่าน** → กด **เข้าสู่ระบบ**
2. เปลี่ยนรหัสผ่านตัวเอง: กดที่ชื่อตัวเองมุมล่างซ้าย → **บัญชีของฉัน**

---

## หน้าที่หลักของคุณ

### 1) รับวัสดุเข้าคลัง (เมนู รับวัสดุ)
**กรณีมีใบสั่งซื้อ (PO):**
1. กดสร้างการรับ → เลือก **คลัง** และ **ใบสั่งซื้อ (PO)**
2. ระบบแสดงเฉพาะรายการที่ยังค้างรับ พร้อมจำนวนคงเหลือ
3. กรอก **จำนวนที่รับจริง** (เป็นจำนวนเต็ม, **รับเกินยอดสั่งไม่ได้**)
4. บันทึก → ของเข้าสต๊อก และสถานะ PO อัปเดตอัตโนมัติ (รับบางส่วน/รับครบ)

**กรณีไม่มี PO:**
- เลือก **ซัพพลายเออร์** แล้วเพิ่มรายการวัสดุเอง (สแกน/พิมพ์รหัส) พร้อมจำนวนและราคาทุน

### 2) จ่ายวัสดุตามคำขอเบิก (เมนู คำขอเบิกวัสดุ)
1. เปิดแถบ **"อนุมัติแล้ว"** — คือคำขอที่ผ่านการอนุมัติ รอจ่ายของ
2. เปิดคำขอ → กด **จ่ายวัสดุ**
3. ระบบตัดสต๊อกออกจากคลังทันที
   - ถ้าของพอ → สถานะ **"จ่ายแล้ว"**
   - ถ้าของไม่พอ → จ่ายได้เท่าที่มี (**"จ่ายบางส่วน"**) ระบบบันทึกยอดที่ขาด — ใส่หมายเหตุได้

### 3) โอนย้ายระหว่างคลัง (เมนู โอนย้ายระหว่างคลัง)
1. กดสร้างใบโอน → เลือก **คลังต้นทาง** และ **คลังปลายทาง**
2. เพิ่มรายการวัสดุ (สแกน/พิมพ์รหัส) + จำนวน (จำนวนเต็ม)
3. บันทึก → ระบบตัดของออกจากต้นทางและเพิ่มเข้าปลายทางพร้อมกัน
   *(ต้นทุนของเดิมถูกยกยอดติดไปด้วย — เป็นการย้ายมูลค่า ไม่ใช่ตีราคาใหม่)*

### 4) นับสต๊อก / ปรับยอด (เมนู นับสต๊อก)
1. กดเริ่มนับ → เลือก **คลัง**
2. ระบบสร้างใบนับของวัสดุที่มีในคลังนั้น พร้อมแสดง **ยอดตามระบบ**
3. เดินนับจริง แล้วกรอก **จำนวนจริง** ที่นับได้ในแต่ละรายการ (ค่าเริ่มต้น = ยอดระบบ)
4. รายการไหนไม่ตรงต้อง **ระบุเหตุผล** (บังคับ)
5. บันทึก → ระบบปรับยอดเฉพาะรายการที่ต่าง

### 5) ดูแลข้อมูลวัสดุและหมวดหมู่
- **Materials:** เพิ่ม / แก้ไข / ปิดการใช้งานวัสดุ (การปิดเป็น soft delete — ข้อมูลเดิมยังอยู่)
- **หมวดหมู่วัสดุ:** สร้าง / เปลี่ยนชื่อ / ลบ (ลบไม่ได้ถ้ายังมีวัสดุใช้หมวดนั้นอยู่)

---

## ข้อควรรู้
- **จำนวนวัสดุต้องเป็นจำนวนเต็มเสมอ** ทุกช่อง (รับ/จ่าย/โอน/นับ)
- **สต๊อกติดลบไม่ได้** — จ่าย/โอนเกินยอดคงเหลือไม่ได้
- ทุกการเคลื่อนไหวถูกบันทึกเป็นบัญชีเดินสต๊อกที่ย้อนแก้ไม่ได้ (ตรวจสอบย้อนกลับได้)
- ทุกเอกสารมีเลขที่รันอัตโนมัติ (GR- รับของ, TR- โอน, SC- นับ) และพิมพ์ PDF ได้
- หากบัญชีคุณเป็นระดับ STAFF จะไม่เห็นราคาทุน — ทำงานคลังได้ตามปกติ

---

## คำถามที่พบบ่อย
**รับของเกินยอดสั่งไม่ได้?** ถูกต้อง ระบบกันไว้ ถ้าของมาเกินจริงให้ประสานฝ่ายจัดซื้อแก้ PO
**จ่ายของแล้วผิด?** สต๊อกตัดแล้วแก้ย้อนไม่ได้โดยตรง ต้องใช้การรับเข้า/นับสต๊อกปรับยอด — ประสานผู้ดูแลระบบ
**นับสต๊อกทำไมบางวัสดุไม่ขึ้น?** ใบนับแสดงเฉพาะวัสดุที่มีสต๊อกในคลังนั้น
