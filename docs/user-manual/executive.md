# คู่มือสำหรับ ผู้บริหาร / ผู้จัดการ (Executive / Manager)

> บทบาทของคุณ: **ดูภาพรวมเชิงบริหาร ต้นทุน/มูลค่า และจัดการโครงการ**
> เมนูที่คุณใช้: Dashboard · รายงาน (รวมการเงิน) · โครงการ · และเมนูปฏิบัติงานทั่วไป · บัญชีของฉัน
> สิทธิ์ข้อมูล: ระดับ **MANAGER** — **เห็นต้นทุนและมูลค่าได้** (ต่างจากพนักงานทั่วไป)

---

## เริ่มต้นใช้งาน
1. เปิดเว็บระบบ → กรอก **อีเมล** และ **รหัสผ่าน** → กด **เข้าสู่ระบบ**
2. เปลี่ยนรหัสผ่านตัวเอง: กดที่ชื่อตัวเองมุมล่างซ้าย → **บัญชีของฉัน**

---

## หน้าที่หลักของคุณ

### 1) Dashboard เชิงบริหาร
หน้าแรกสรุปภาพรวม รวมข้อมูลเชิงมูลค่าที่พนักงานทั่วไปไม่เห็น

### 2) รายงาน (เมนู รายงาน)
**รายงานทั่วไป**
- สต๊อกคงเหลือ · วัสดุต่ำกว่าจุดสั่งซื้อ · กิจกรรมการเบิกจ่าย · กิจกรรมการรับวัสดุ · สัดส่วนสต๊อกตามไซต์

**รายงานการเงิน** (เฉพาะผู้บริหาร/ผู้จัดการ)
- **มูลค่าสต๊อก** — คำนวณต้นทุนถัวเฉลี่ยถ่วงน้ำหนักจริงจากประวัติทั้งหมด (ไม่ใช่ค่าเฉลี่ยแบบง่าย)
- **ประวัติการเบิกจ่าย** — พร้อมมูลค่าที่เบิกไป
- **สรุปการเงินตามไซต์งาน** — ยอดคงเหลือ / เบิกใช้ / รับเข้า แยกตามไซต์

> กรองช่วงเวลา/เงื่อนไขได้ในแต่ละรายงาน และดูรายไซต์ได้

### 3) จัดการโครงการ (เมนู โครงการ)
- **ดูรายการโครงการ** พร้อมงบประมาณ มูลค่าสัญญา ลูกค้า และคลังของโครงการ
- **สร้างโครงการใหม่:** ระบบสร้างโครงการ + ลูกค้า (เลือกเดิมหรือสร้างใหม่ในตัว) + คลังไซต์ ให้พร้อมกันในครั้งเดียว
- **เปลี่ยนสถานะโครงการ:**
  ```
  วางแผน → กำลังดำเนินการ → เสร็จสิ้น
                    └──────────→ ยกเลิก
  ```
- **ปิดโครงการปลอดภัย:** เมื่อกด "เสร็จสิ้น" หรือ "ยกเลิก" ระบบจะให้ **ยืนยันอีกครั้ง** ก่อน (กดครั้งเดียวไม่ปิดทันที กันกดพลาด)

---

## ข้อควรรู้
- **โครงการที่ปิด/ยกเลิกแล้ว จะเบิกวัสดุใหม่เข้าโครงการนั้นไม่ได้** (การปิดเป็นการหยุดรับเบิกใหม่ ไม่ได้ลบข้อมูล)
- **การเปิดโครงการที่ปิดไปแล้วกลับมา (Reopen) ทำได้เฉพาะ ADMIN** — หากปิดผิดให้แจ้งผู้ดูแลระบบ
- คุณเห็นเมนูโครงการเพราะสิทธิ์ระดับ MANAGER (พนักงานทั่วไปไม่เห็น)
- คุณ **ไม่มีสิทธิ์จัดการผู้ใช้** (เป็นของ ADMIN)

---

## คำถามที่พบบ่อย
**ปิดโครงการผิด?** แจ้ง ADMIN ให้กด Reopen — เปิดกลับได้ทั้งหมด ไม่มีข้อมูลหาย
**ตัวเลขมูลค่าสต๊อกมาจากไหน?** คำนวณจากประวัติการรับ-จ่ายจริงทุกครั้งด้วยต้นทุนถัวเฉลี่ยถ่วงน้ำหนัก จึงสะท้อนต้นทุนจริง
**สร้างโครงการแล้วต้องสร้างคลังแยกอีกไหม?** ไม่ต้อง ระบบสร้างคลังไซต์ให้พร้อมกับโครงการอัตโนมัติ
