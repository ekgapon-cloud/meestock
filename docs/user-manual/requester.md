# คู่มือสำหรับ ผู้ขอเบิกวัสดุ (Requester)

> บทบาทของคุณ: **ขอเบิกวัสดุไปใช้หน้างาน** และติดตามสถานะคำขอ
> เมนูที่คุณใช้: Dashboard · Materials · คำขอเบิกวัสดุ · รายงาน · บัญชีของฉัน

---

## เริ่มต้นใช้งาน
1. เปิดเว็บระบบ → กรอก **อีเมล** และ **รหัสผ่าน** ที่ได้รับ → กด **เข้าสู่ระบบ**
2. ออกจากระบบ: ปุ่ม **ออกจากระบบ** มุมล่างซ้าย
3. เปลี่ยนรหัสผ่านตัวเอง: กดที่ชื่อตัวเองมุมล่างซ้าย → **บัญชีของฉัน** → กรอกรหัสเดิม + รหัสใหม่ (อย่างน้อย 8 ตัว) + ยืนยัน

---

## หน้าที่หลักของคุณ

### 1) ค้นหาวัสดุ (เมนู Materials)
- ค้นหาด้วยชื่อ/รหัสวัสดุ หรือ **สแกนบาร์โค้ด** ได้
- ดูได้ว่าวัสดุมีอะไรบ้าง อยู่หมวดไหน
- *หมายเหตุ:* คุณจะ**ไม่เห็นราคาทุน**ของวัสดุ (เป็นไปตามสิทธิ์ระดับ STAFF)

### 2) สร้างคำขอเบิกวัสดุ (เมนู คำขอเบิกวัสดุ → สร้าง)
1. กด **สร้างคำขอเบิก**
2. เพิ่มรายการวัสดุ ได้ 3 วิธี:
   - **สแกนบาร์โค้ด** ด้วยเครื่องสแกน
   - **พิมพ์รหัสวัสดุ** แล้วกดเพิ่ม
   - **สแกนด้วยกล้องมือถือ** (ปุ่มกล้อง)
3. ใส่ **จำนวน** ที่ต้องการ (ต้องเป็น**จำนวนเต็ม** — ใส่ทศนิยมไม่ได้)
4. ระบุโครงการ/ไซต์ที่จะใช้
5. กด **ส่งคำขอ** → คำขอจะอยู่สถานะ **"รออนุมัติ"**

### 3) ติดตามสถานะคำขอ
เปิดเมนู **คำขอเบิกวัสดุ** จะเห็นแถบสถานะ:
| สถานะ | ความหมาย |
|-------|---------|
| รออนุมัติ | รอผู้อนุมัติพิจารณา |
| อนุมัติแล้ว | รอเจ้าหน้าที่คลังจ่ายของ |
| จ่ายแล้ว | ได้รับของครบแล้ว |
| จ่ายบางส่วน | ได้ของบางส่วน (ของในคลังไม่พอ ระบบบันทึกยอดที่ขาดไว้) |
| ปฏิเสธ | ผู้อนุมัติไม่อนุมัติ |

- ถ้าคำขอค้างนานจะมีป้าย **"เกินกำหนด"** เตือน
- พิมพ์ใบเบิกเป็น PDF ได้จากหน้ารายละเอียดคำขอ

---

## ข้อควรรู้
- **คุณอนุมัติคำขอของตัวเองไม่ได้** — ต้องให้ผู้อนุมัติ (Approver) คนอื่นเป็นผู้อนุมัติ
- ถ้าโครงการถูกปิด/ยกเลิกแล้ว จะ**เบิกวัสดุใหม่เข้าโครงการนั้นไม่ได้**
- คุณจะไม่เห็นเมนู โครงการ / ผู้ใช้งาน และปุ่มอนุมัติ — เป็นเรื่องปกติตามสิทธิ์

---

## คำถามที่พบบ่อย
**ขอเบิกแล้วของไม่พอ?** ระบบจ่ายเท่าที่มี (จ่ายบางส่วน) และบันทึกยอดที่ขาดไว้ ให้ติดตามหรือขอเบิกเพิ่มภายหลัง
**ทำไมไม่เห็นราคา?** บัญชีระดับ STAFF ถูกออกแบบให้ไม่เห็นต้นทุน
**ลืมรหัสผ่าน?** แจ้งผู้ดูแลระบบ (Admin) ให้ตั้งรหัสใหม่ให้
